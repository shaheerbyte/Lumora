# Lumora

**Lumora** is a privacy-first, fully offline trading journal designed for traders who want to track their trades, analyze performance, and improve their strategies â€” all without relying on the cloud.  

Lumora is built for **Android** initially, with plans to expand to **iOS, Windows, and Linux**, making it a universal cross-platform trading tool.  

---

## Features

- **100% offline:** All data is stored locally; no internet required.  
- **Private and secure:** No tracking, analytics, or cloud sync.  
- **Comprehensive trade logging:** Entry/Exit prices, Stop Loss/Take Profit, results, risk-reward ratio, setup, emotions, mistakes, and notes.  
- **Review support:** Weekly and monthly summaries for performance analysis.  
- **Future-proof architecture:** Designed for cross-platform expansion.

---

## Tech Stack

- **Android:** Kotlin + Room (SQLite)  
- **Future cross-platform:** Flutter planned for iOS, Windows, and Linux releases.  
- **UI:** Material Design for clean and responsive interface.
